**Datasets used**
* Pittsburgh Public Schools Enrollment by Neighborhood, School, and Feeder Pattern 
https://data.wprdc.org/dataset/pittsburgh-public-schools-enrollment. This dataset includes the total number of students enrolled in each neighborhood by education level
* 2009-13 and 2019-23 American Community Survey estimates for City of Pittsburgh neighborhoods: https://data.wprdc.org/dataset/2009-13-and-2019-23-american-community-survey-estimates-for-city-of-pittsburgh-neighborhoods. This dataset contains population estimates for neighborhoods in Pittsburgh
* POGOH Station Locations - October 2025: https://data.wprdc.org/dataset/station-locations (This Dataset contains all POGOH Bike share Station Locations in Allegheny County for the month of October in 2025.)
